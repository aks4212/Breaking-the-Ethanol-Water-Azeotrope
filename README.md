# Breaking the Ethanol-Water Azeotrope 

This repository contains the thermodynamic models, phase equilibrium data, and separation process calculations for breaking a complex azeotropic mixture. The project explores advanced separation methodologies, such as extractive or pressure-swing distillation, to bypass standard vapor-liquid equilibrium limitations.

---

## Project Structure

* **`Phase_Diagrams/`**: Contains scripts and plotted ternary phase diagrams, illustrating residue curve maps and distillation boundaries.
* **`VLE_Data/`**: Datasets containing non-ideal Vapor-Liquid Equilibrium (VLE) parameters and activity coefficient modeling.
* **`Calculations/`**: Computational scripts used for stage-by-stage mass/energy balances, operating line determination, and minimum reflux ratio calculations.

---

## Thermodynamic Concepts & Methodologies

* **Azeotropic & Extractive Distillation:** Designing separation sequences using heavy/light entrainers to alter relative volatilities and cross distillation boundaries.
* **Phase Equilibrium Analysis:** Utilizing activity coefficient models (e.g., NRTL, UNIQUAC) to accurately predict non-ideal mixture behavior and calculate phase splits.
* **Ternary Phase Diagrams:** Mapping multi-component mixtures to identify optimal feed locations, solvent-to-feed ratios, and product compositions.
* **Process Optimization:** Iterative calculation of operating lines to minimize theoretical stages and optimize column thermal duties.
